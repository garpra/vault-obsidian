---  
```

Fleeting Note – 2025-04-29-01  
  
Linux bukan satu sistem operasi tunggal, tetapi kumpulan distribusi yang berbagi kernel yang sama. Kernel adalah inti dari sistem operasi, bertanggung jawab untuk komunikasi antara perangkat keras dan perangkat lunak. Setiap distribusi (seperti Ubuntu, Arch, Fedora) memiliki tujuan, filosofi, dan pengelolaan paket yang berbeda.  
  
Catatan: Mungkin perlu eksplorasi lebih lanjut tentang bagaimana filosofi masing-masing distro memengaruhi penggunaannya (mis. Arch = KISS + kendali penuh vs. Ubuntu = kemudahan akses + kestabilan).  

```
---  
  
Ingat bahwa fleeting note bersifat cepat, mentah, dan pribadi—fokus pada menangkap ide secepat mungkin tanpa menyunting terlalu banyak. Nantinya, ini bisa dikembangkan menjadi literature note atau permanent note dalam sistem Zettelkasten.
