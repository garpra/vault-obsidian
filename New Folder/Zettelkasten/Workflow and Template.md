
## WORKFLOW ZETTELKASTEN DI OBSIDIAN

1. Tangkap Ide Cepat (Fleeting Note)
	- Gunakan saat sedang membaca, berpikir, atau mendengar ide menarik.
	- Tuliskan dengan bebas, cepat, dan tidak perlu rapi.
	- Review secara rutin (misalnya setiap malam atau akhir minggu).

2. Ubah jadi Literature Note
	- Bila ide berasal dari sumber tertentu, rangkum dengan singkat.
	- Sertakan referensi, kutipan, dan catatan pribadi.
	- Gunakan saat membaca serius (buku, artikel, kuliah, dll).

3. Tulis Permanent Note
	- Buat berdasarkan hasil refleksi dari satu atau lebih Literature/Fleeting Note.
	- Gunakan bahasa sendiri, dan buat agar catatan bisa berdiri sendiri.
	- Hubungkan dengan catatan lain bila relevan ([[...]] di Obsidian).

---

## TEMPLATE FLEETING NOTE

```
Filename: fleeting-{{date}}-{{time}}

# Fleeting Note – {{date}}-{{time}}

## Isi Cepat
Tuliskan ide, pertanyaan, pemikiran secara bebas di sini...

## Konteks (opsional)
- Aktivitas: (misal: membaca artikel, ngobrol, ide muncul saat ngopi)
- Perlu ditindaklanjuti? Y/N

```

---

## TEMPLATE LITERATURE NOTE

```
Filename: lit-{{judul atau topik}}

# Literature Note – {{judul atau topik}}

## Sumber
- Judul: 
- Penulis:
- Tanggal akses:
- Tautan (jika ada):

## Ringkasan
Tuliskan ringkasan isi sumber dengan kata-kata sendiri.

## Kutipan Penting (opsional)
> “Kutipan dari sumber, jika relevan.”

## Catatan Pribadi
Apa yang menurutmu penting? Relevansi dengan topik lain?

## Tindak Lanjut
- [ ] Butuh diubah ke Permanent Note?
- [ ] Hubungkan dengan topik lain?
```

---

## TEMPLATE PERMANENT NOTE

```
Filename: Z-{{topik-unik}} (misalnya: Z-wayland-vs-x11)

# {{Judul Topik}}

## Ide Inti
Tuliskan satu ide per catatan, dengan jelas dan padat.

## Penjelasan
Tuliskan dengan kata-kata sendiri, untuk menjelaskan ide ini. Bayangkan kamu sedang menjelaskannya ke orang lain. Sertakan contoh jika perlu.

## Hubungan
- Terkait dengan: [[lit-judul]] | [[Z-topik-lain]]
- Konsep lebih besar: [[filosofi-desain]] atau [[evolusi-linux]]
- Lawan/kontras: [[Z-monolitik-vs-modular]]

## Tags
#permanent #linux #grafis #wayland #x11

```
