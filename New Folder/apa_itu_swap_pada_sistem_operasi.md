### Poin Penting
**Swap** pada sistem operasi adalah ruang di disk yang digunakan sebagai perpanjangan memori (RAM) ketika RAM fisik sudah penuh. Ketika sistem kehabisan RAM, data yang tidak aktif dari RAM dipindahkan ke ruang swap agar RAM bisa digunakan oleh proses yang lebih aktif.

Swap bisa berada dalam dua bentuk:

1. **Partisi Swap**: Partisi khusus di hard drive yang dialokasikan untuk swap.
2. **File Swap**: File yang berada di sistem file utama yang digunakan sebagai swap.

Meskipun swap dapat membantu sistem untuk tetap berjalan ketika RAM penuh, mengandalkan swap terlalu sering dapat menurunkan performa karena kecepatan disk lebih lambat dibandingkan RAM. Oleh karena itu, swap digunakan sebagai cadangan, dan peningkatan RAM lebih disarankan jika sistem sering menggunakan swap.
