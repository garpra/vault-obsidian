
## highest-lowest-javascript

**tags:** #type/log #topic/programming #topic/javascript

### Catatan
Saat saya sedang mengerjakan coding challenge dari website codewars yang bernama Highest and Lowest, saya mengalami kesulitan pada Math.max, saya tahu function ini untuk mencari angkat terbesar tetapi setiap dicoba selalu error, Akhirnya saya pun kesal dan copy paste codingan ke prompt chatgpt dan diberi tahu bahwa Math.max argumentnya tidak boleh berbentuk array tetapi haru individual number

```javascript

// Harus berbetuk individual number
Math.max(1,2,3,4) 

// Malah saya isi berbetuk array 
Math.max([1,2,3,4])

```
