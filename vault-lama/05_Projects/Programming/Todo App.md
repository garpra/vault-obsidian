## Judul : Todo App
### Tujuan
Aplikasi web dibuat dengan tujuan pengingat dan pencatat tugas harian atau mendatang untuk membantu pengguna mengelola tugas lebih efisien, manajemen waktu, dan memudahkan pengontrolan atas tugas-tugas yang akan dikerjakan.

### Fitur
- Halaman Home
	- Tombol **"Add Todo"** untuk menambahkan daftar tugas baru
	- Daftar tugas yang sudah pernah dibuat
	- Tombol **"Hapus"** untuk menghapus salah satu daftar tugas
	- Tombol **"Edit"** untuk mengedit salah satu daftar tugas
- Halaman Tambah todo
	- Input Judul Todo
	- Tombol **"+"** untuk menambahkan tugas baru ke daftar
	- Checkbox untuk mengecek apakah tugas sudah dikerjakan atau belum
	- Tombol **"Delete"** untuk menghapus tugas dari daftar
	- Tombol **"Save"** untuk menyimpan daftar tugas
- Halaman Edit todo
	- Edit Judul todo
	- Tombol **"+"** untuk menambahkan tugas baru ke daftar
	- Checkbox untuk mengecek apakah tugas sudah dikerjakan atau belum
	- Tombol **"Delete"** untuk menghapus tugas dari daftar
	- Tombol **"Edit"** untuk menyimpan perubahan daftar tugas
- Halaman Detail todo
	- Menampilkan detail dari daftar tugas 

### Alat dan Teknologi 
- HTML
- Javascript
- NPM
- Font Awesome
- Tailwindcss

### Alur Aplikasi
- **Halaman Home**
	- Saat pengguna pertama membuka website, mereka akan diarahkan ke halaman utama.
	- Di halaman ini, pengguna dapat melihat list daftar tugas yang sudah dibuat yang tersimpan di localStorage
	- Pengguna juga bisa menambahkan daftar tugas baru dengan menekan tombol **"Add Todo"**
	- Dan juga mengedit daftar tugas yang sudah dibuat dengan menekan tombol **"Edit"**
- **Halaman Tambah Todo**
	- Di halaman ini pengguna bisa mengisikan judul untuk daftar tugas dan menambahkan tugas baru dengan menekan tombol **"+"**.
	- Pengguna juga bisa mengedit dan menghapus tugas yang sudah dibuat
	- Untuk menyimpan daftar tugas pengguna bisa menekan tombol **"Save"** dan daftar akan tersimpan ke localStorage
- **Halaman Edit Todo**
	- Di halaman ini pengguna bisa mengedit daftar tugas yang sudah dibuat, tampilan sama seperti **Halamana Tambah Todo**
	- Untuk menyimpan perubahan pengguna bisa menekan tombol **"Edit"** dan tersimpan di localStorage
- **Halaman Detail Todo**
	- Di halaman ini pengguna dapat melihat detail daftar tugas yang ingin diselesaikan
	- Pengguna juga dapat checklist checkbox jika tugas sudah selesai dikerjakan
