## Judul : BMI Calculator
### Tujuan
Aplikasi web yang dibuat dengan tujuan sebagai alat penghitung berat badan ideal menurut standar BMI (Body Mass Index). Dengan aplikasi ini pengguna bisa mencari tahukan apakah berat badannya sudah ideal, underweight, atau overweight

### Fitur
- Halaman Utama
	- Input tinggi badan pengguna
	- Input berat badan pengguna
	- Tombol **"Calculate"** untuk menghitung BMI sesuai dengan input
	- Tombol **"Reset"** untuk mereset input dari pengguna
	- Menampilkan hasil dari BMI beserta kategorinya

### Alat dan Teknologi 
- HTML
- CSS
- Javascript
- Tailwindcss

### Alur Aplikasi
- Saat pertama kali mengakses website, pengguna akan langsung melihat halaman utama.
- Untuk dapat menghitung BMI pengguna harus menginputkan data tinggi dan berat badan pengguna
- Setelah menginputkan tinggi dan berat badan pengguna bisa langsung menghitung BMI dengan menekan tombol **"Calculate"**.
- Sistem akan menghitung BMI dan akan ditampilkan di halaman beserta kategorinya
- Pengguna juga bisa mengatur ulang input dan hasil BMI dengan menekan tombol **"Reset"**.
