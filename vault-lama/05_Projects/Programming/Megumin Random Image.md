## Judul : Megumin Random Image
### Tujuan
Aplikasi web ini dibuat dengan tujuan mempelajari penggunaan API, yang berguna untuk random gambar karakter **"Megumin"** dari anime Konosuba dan pengguna juga dapat mendownload gambar hasil random anda, gambar karakter Megumin didapat dari waifu.pic API.

### Fitur
- Halaman Utama
    - Menampilkan gambar megumin secara acak
    - Penjelasan Singkat mengenai cara penggunaan aplikasi
    - Tombol **"Random"** untuk mengacak gambar dari API
    - Fitur download gambar dengan menekan gambar
- Halaman Gallery
    - Tombol **"Random All"** untuk mengacak banyak gambar sekaligus
    - Daftar gambar Megumin dalam grid
- Halaman About
	- Informasi tentang pembuat aplikasi

### Alat dan Teknologi 
- HTML
- Tailwindcss
- Javascript
- waifu.pics (API)
- Flowbite

### Alur Aplikasi
- **Halaman Utama (Homepage)**
    - Saat pengguna pertama kali membuka website, mereka akan diarahkan ke halaman utama.
    - Di halaman ini, pengguna dapat menekan tombol **"Random"** untuk menampilkan satu gambar Megumin secara acak.
    - Pengguna juga dapat mengunduh gambar yang sedang ditampilkan dengan menekan langsung gambar tersebut.
    
- **Halaman Gallery**
    - Di halaman ini, pengguna dapat menampilkan beberapa gambar Megumin sekaligus dengan menekan tombol **"Random All"**.
    - Gambar-gambar akan ditampilkan dalam format grid agar mudah dilihat dan dinavigasi.
    
- **Halaman About**
    - Halaman ini memuat informasi singkat tentang developer, serta tujuan dari pembuatan proyek.
