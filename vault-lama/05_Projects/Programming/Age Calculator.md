## Judul : Age Calculator
### Tujuan
Aplikasi web ini dibuat dengan tujuan sebagai kalkukator umur yang dapat menghitung umur dari pengguna dengan memasukkan tanggal lahir, dan sistem akan menghitung umur dari tanggal lahir dan tanggal saat ini 

### Fitur
- Halaman Utama
	- Menampilkan umur dari pengguna dengan format "Tahun Bulan Hari"
	- Input Tanggal untuk memasukkan tanggal lahir dari perngguna
	- Tombol **"Hitung"** untuk menghitung umur sesuai dengan tanggal lahir

### Alat dan Teknologi 
- HTML
- CSS
- Javascript

### Alur Aplikasi
- Saat pertama kali mengakses website, pengguna akan langsung melihat halaman yang menampilkan umur anda.
- Untuk bisa menghitung umur pengguna harus memasukkan tanggal lahir pada input date pada halaman ini
- Setelah menginputkan tanggal lahir pengguna bisa menghitung umur dengan menekan tombol **"Hitung"**
