## Judul : Pomodoro Timer
### Tujuan
Aplikasi web ini dibuat dengan tujuan sebagai timer untuk metode pembelajaran pomodoro yang membantu pengguna meningkatkan fokus dan mencegah kelelahan akibat belajar terlalu lama. Teknik pomodoro biasanya dilakukan dengan cara 25 menit fokus dan 5 menit istirahat.

### Fitur
- Halaman Utama 
	- Menampilkan timer untuk metode pomodoro
	- Tombol **"Start"** untuk memulai pomodoro timer
	- Tombol **"Pause"** untuk menghentikan sementara pomodoro timer
	- Tombol **"Reset"** untuk memulai kembali dari awal pomodoro timer

### Alat dan Teknologi 
- HTML
- CSS
- Javascript

### Alur Aplikasi
- Saat pertama kali mengakses website, pengguna akan langsung melihat halaman yang menampilkan timer Pomodoro.
- Di halaman ini, terdapat tombol **"Start"** yang digunakan untuk memulai timer.
- Timer akan menghitung waktu selama 25 menit untuk periode fokus dan 5 menit untuk periode istirahat.
- Setelah tombol **"Start"** ditekan, timer Pomodoro akan mulai menghitung waktu, dan tombol **"Start"** akan berubah menjadi **"Pause"** dan **"Reset"**.
- Pengguna dapat menekan tombol **"Pause"** untuk menghentikan timer sementara, dan tombol **"Reset"** untuk memulai kembali timer dari awal.
- Ketika waktu fokus selesai, pengguna akan diberi pemberitahuan untuk beristirahat, dan setelah periode istirahat selesai, timer kembali ke waktu fokus.
