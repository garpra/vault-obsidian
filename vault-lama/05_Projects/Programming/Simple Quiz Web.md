## Judul : Simple Quiz Web
### Tujuan
Aplikasi web ini dibuat dengan tujuan sebagai media quiz dimana pengguna bisa mengerjakan quiz sesuai dengan mata pelajaran yang dipilih.

### Fitur
- Halaman Home
	- Tombol **"Start"** untuk memulai mengerjakan quiz
	- Dropdown **"Selection"** untuk memilih mata pelajaran yang ingin diikuti
- Halaman Quiz
	- Menampilkan pertanyaan dari mata pelajaran yang dipilih secara acak
	- Menampilkan pilihan ganda sesuai dengan pertanyaan
	- Menampilkan waktu untuk setiap pertanyaan 
- Halaman hasil nilai
	- Menghitung dan menampilkan hasil nilai dari quiz yang sudah dikerjakan
	- Tombol **"Kembali"** untuk kembali ke halaman utama

### Alat dan Teknologi 
- HTML  
- CSS
- JavaScript
- JSON

### Alur Aplikasi
- **Halama Home**
	- Saat pertama kali mengakses website, pengguna akan diarahkan ke halaman home.
	- Di halaman ini pengguna bisa memilih mata pelajaran yang ingin dikerjakan dalam quiz pada dropdown.
	- Setelah memilih mata pelajaran pengguna bisa menekan tombol **"Start"** untuk memulai quiz.
- **Halaman Quiz**
	- Untuk mengerjakan satu kuis pengguna diberi waktu 10 detik.
	- Halaman ini akan menampilkan pertanyaan dari mata pelajaran yang dipilih beserta pilihan ganda
	- Pengguna harus memilih pilihan ganda sebelum waktu habis, jika tidak maka pengguna akan dibawa ke pertanyaan selanjutnya.
- **Halaman Hasil Nilai**
	- Sistem akan menghitung hasil nilai setelah pengguna selesaik mengerjakan semua soal dan hasil nilai akan ditampilkan
	- Pengguna bisa kembali ke halaman home dengan menekan tombol **"Kembali"**
