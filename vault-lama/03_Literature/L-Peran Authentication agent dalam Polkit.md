## Judul: Peran Authentication agent dalam Polkit

### Ringkasan
Authentication agent ini diperlukan untuk validasi pengguna saat melakukan proses yang memerlukan akses root. Kebanyakan Desktop Environtment (DE) seperti KDE, XFCE4, GNOME memiliki authentication agent bawaan contohnya `polkit-kde-agent`, `polkit-gnome`, atau `lxqt-policykit`.
